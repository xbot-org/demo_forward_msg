# demo_forward_msg

基于 xbot 实现微信消息监控转发

## 使用

先配置好 xbot 的配置如下，并运行
```
CALLBACK_URL=http://127.0.0.1:5678
SECRET=xxx
PORT=5557
```

clone 代码
```
git clone https://github.com/xbot-org/demo_forward_msg.git
cd demo_forward_msg
```

运行此服务 `php -S localhost:5678`
